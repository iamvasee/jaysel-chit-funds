# Jaysel Chit Funds Landing Page

A modern, responsive landing page for Jaysel Chit Funds built with Next.js 14 and Tailwind CSS.

## Features

- Modern UI with Hanken Grotesk font
- Responsive design for all devices
- Interactive components with smooth animations
- Optimized images and assets
- SEO friendly

## Tech Stack

- Next.js 14
- React
- Tailwind CSS
- TypeScript

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/iamvasee/jaysel-chit-funds.git
```

2. Install dependencies:
```bash
cd jaysel-chit-funds
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `/app` - Next.js app router pages and layouts
- `/components` - React components
- `/public` - Static assets (images, icons)
- `/styles` - Global styles and Tailwind configuration
